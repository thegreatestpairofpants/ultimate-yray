How to install the modpack

  1. Download the fabric launcher installer from the link above
  2. Install fabric for 1.18.2
  3. Download the mods from the link above
  4. Go to your .minecraft folder via %appdata%/.minecraft
  5. Create a folder named "mods" if there already is one, empty it
  6. Extract/move the all the mods in the zip to the mods folder
