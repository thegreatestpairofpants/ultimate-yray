How to install the modpack

  1. Download the fabric launcher installer
  2. Install fabric for 1.19.2
  3. Download the mods from the media fire link above
  4. Go to your .minecraft folder via typing %appdata%/.minecraft as file location into file explorer
  5. Create a folder named "mods". If a mod folder already exists, then empty it
  6. Extract/move the all the mods in the .zip you downloaded into the mods folder
